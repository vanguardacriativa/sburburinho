# 🐰✨ Páscoa Cosmos 2026 — Burburinho Café

> Cardápio digital que transforma produto em desejo.

---

## Sobre o projeto

Desenvolvimento do **cardápio digital da Páscoa Cosmos 2026 do Burburinho Café** — um espaço que surgiu no Pirambu, em Fortaleza, com a proposta de tornar a boa gastronomia acessível dentro da própria comunidade.

Mais do que listar produtos, a experiência foi pensada para despertar interesse, facilitar a decisão e valorizar cada criação artesanal.

---

## Objetivo

- Destacar os produtos sazonais de Páscoa com identidade visual forte
- Criar navegação simples, intuitiva e encantadora
- Gerar desejo através da apresentação visual e dos textos
- Facilitar o processo de escolha e conversão do cliente

---

## Tecnologias

- HTML5 · CSS3 · JavaScript
- Google Fonts (Cormorant Garamond + Poppins)
- Deploy via Vercel ou GitHub Pages
